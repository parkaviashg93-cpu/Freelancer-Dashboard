# Freelancer-Dashboard
Project Overview:
This project focuses on analyzing freelancer performance data and creating an interactive dashboard that visualizes key metrics such as activity status, experience, project count, gender-wise hourly rates, and engagement patterns.
The dashboard transforms raw freelancer data into meaningful insights that help understand productivity trends and earning behaviors.

Objectives:
Examine the distribution of Active vs Inactive freelancers.
Compare hourly rates across genders.
Study the relationship between experience and number of completed projects.
Understand correlations between hourly rates and project counts.
Develop an interactive dashboard using slicers and visual charts.

Dataset Description:
Column Name                 	Description
Freelancer_ID	                Unique ID of freelancer
Is Active	                    YES / NO status
Name	                        Freelancer's full name
Gender	                      Male / Female
Age	                          Age of freelancer
Hourly Rate ($)	              Charges per hour
Years of Experience	          Total work experience
No. of Projects	              Count of completed projects
Project Status	              High / Medium / Low
Duration_months	              Time taken for recent projects

Dashboard Elements:
1. Slicers:
Active Status
Experience
Gender
2. Visuals:
Bar Chart: Count of Freelancers (Active/Inactive)
Pie Chart: Distribution of Experience Levels
Line Chart: Hourly Rate by Gender
